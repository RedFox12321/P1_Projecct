funções:
 M->Mostrar todos os dados


 (done)     I->Inserir portatil  // Inserir a informação referente a um portatil.
 (done)     A->Alterar dados de um portatil //switch com o dado a trocar
 (    )     D->Adicionar dados sobre a avaria/reparacao de um portatil //Adicionar avaria de um portatil
 (    )     P->Mostrar avarias  //Lista as avarias dos portateis que existem
 (done)     M->Mostrar dados dos portateis //Lista os portateis e todas as suas informacoes


 (    )     A->Adicionar requisicao // Adicionar uma requisicao de um portatil
 (    )     R->Renovar a requisicao // Aumentar a data final de uma requisicao
 (    )     E->Eliminar requisicao // Remover uma requisicao existente
 (    )     D->Fazer a Devolucao // Mudar o estado de uma requisicao para concluido(0)
 (    )     P->Pesquisar dados sobre uma requisicao // Pesquisar sobre uma requisicao por numero ou nome (not case sensitive)
 (    )     M->Mostrar dados das requisicoes //Lista todas as requisicoes e todas as suas informacoes


 (done)     B->Gravar em ficheiro binário //grava ficheiro binario em portateis.dat
 (done)     L->Gravar em ficheiro binário e criar ficheiro log_texto //grava ficheiro binario em portateis.dat e cria/recria um log_file/ficheiro texto "portateis.txt"
 (done)     C->Carregar ficheiro binário //carrega ficheiro binario portateis.dat
